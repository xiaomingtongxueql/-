# 溯鉴历史 (Sujian History) v2.0 - 全维交互版

基于人教版初高中历史教科书的深度交互式历史教育平台。

## 🌟 v2.0 核心特性

- **🇨🇳 中国史全系深度升级**：夏、商、周、秦、汉、三国、魏晋、隋、唐、宋、元、明、清 13个核心断代史页面全部达到研究生级深度标准（2500字+）。
- **🎨 沉浸式水墨体验**：新增 **三国 (Three Kingdoms)** 专属页面，采用魏(红)/吴(蓝)/蜀(绿) 动态水墨配色与阵营博弈图谱。
- **🔗 智能知识网络**：全站 188+ 个话题页（Topic）已通过智能匹配算法挂载了 **"历史深读"** 入口，实现了从微观词条到宏观历史的无缝跳转。
- **📜 极致阅读体验**：
  - **滚动位置记忆**：浏览详情页返回主页时，自动定位到之前的时间轴位置。
  - **动态知识图谱**：基于 D3.js 的力导向图，可视化展示人物、事件、制度的复杂关系。

## 快速开始

### 🚀 部署
只需将 **`dist/`** 文件夹上传至任意静态 Web 服务器（Nginx, GitHub Pages, Vercel）即可，无需 Node.js 环境。

### 🛠️ 开发
如果需要修改内容：
1. 修改 `data/details/` 下的 JSON 数据。
2. 运行 `node scripts/build_detail.js data/details/detail_xxx.json` 重新生成 HTML。
3. 运行 `node scripts/package.js` 更新发布包。
