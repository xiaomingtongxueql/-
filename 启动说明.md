# 🚀 快速启动指南

## 问题说明
原始 `index.html` 由于以下原因可能无法正常显示：
1. **CORS限制**：直接双击HTML文件无法加载外部JSON数据
2. **网络依赖**：需要从CDN加载Tailwind CSS等外部资源

## ✅ 解决方案

### 方案1：独立版（最推荐）
- **文件**：`index-standalone.html`
- **特点**：
  - 完全独立，不依赖任何外部资源
  - 使用内嵌数据，无需JSON文件
  - 内嵌样式，无需CSS框架
  - 可以直接双击打开
- **使用方法**：
  1. 双击 `index-standalone.html`
  2. 立即可以看到18个中国史事件

### 方案2：启动工具
- **文件**：`启动.bat`
- **使用方法**：
  1. 双击 `启动.bat`
  2. 选择 `[1]` 打开独立版（推荐）
  3. 选择 `[2]` 打开测试版
  4. 选择 `[3]` 打开调试版
  5. 选择 `[4]` 启动本地服务器

### 方案3：本地服务器
**需要Python环境**

#### Windows PowerShell:
```powershell
python server.py
# 然后访问 http://localhost:8000
```

#### Windows CMD:
```cmd
python server.py
# 然后访问 http://localhost:8000
```

## ❓ 常见问题

### Q: 双击后空白页面
A: 检查浏览器是否支持，推荐使用 Chrome、Edge、Firefox 最新版

### Q: 看不到事件卡片
A:
1. 检查浏览器控制台（F12）是否有错误
2. 确认使用的是 `index-standalone.html`
3. 尝试使用 `启动.bat` 工具

### Q: 点击标签没有反应
A: 刷新页面重试
